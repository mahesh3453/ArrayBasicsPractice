# Array Basics Practice

A collection of Java programs for practicing core array manipulation, sorting, searching, and custom array problems.

## 📌 Repository Contents

This repository contains implementation examples for common array operations:

### 🔍 Searching & Sorting Algorithms
- **BinarySearch.java**: Implementation of Binary Search on a sorted array.
- **BubbleSort.java**: Standard Bubble Sort algorithm.
- **InsertionSort.java**: Standard Insertion Sort algorithm.

### 🛠️ Common Array Operations
- **RemoveDuplicatesFromArray.java**: Finding and removing duplicates from an array.
- **ExamMain.java**: Main driver/test class for array exam problems.

### 📝 Practice Tasks (Task01 to Task22)
A series of structured practical challenges focusing on real-world array manipulation scenarios:
- **Task01**: Merging two branch sales lists (sequential combination of arrays).
- **Task02**: Generating combined seating from two rows (interleaved/zig-zag combination).
- **Task03 to Task22**: Additional logic building exercises including filtering, reversing, finding min/max values, splitting, shifting, rotating, and managing array contents.

---

## 🚀 How to Run the Programs

Ensure you have the [Java Development Kit (JDK)](https://www.oracle.com/java/technologies/downloads/) installed.

### 1. Compile the file
Navigate to the source root and compile the target Java class. For example:
```bash
javac com/nit/arrays/Task01.java
```

### 2. Run the program
Execute the compiled Java class:
```bash
java com.nit.arrays.Task01
```
